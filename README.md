# boqian
#项目安装
    1.安装vue-cli
        npm install -g vue-cli
    2.初始化项目
        npm init webpack boqian
    3.进入项目
        cd boqian
    4.安装依赖
        npm install
    5.启动项目
        npm run dev
#项目目录结构
    index.html  项目根视图
    static: 静态文件目录
#模板语法

    Vue组件
        包含三个部分
        template  视图
        script   交互
        style    样式
